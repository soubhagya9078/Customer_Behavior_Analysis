📊 Data Analyst Project – End-to-End Analysis

🔍Overview

This project demonstrates a complete data analysis workflow, starting from raw data to actionable insights. It includes data loading, cleaning, SQL querying, and dashboard creation. The goal is to extract meaningful insights and present them in a clear, visual format.


📁 Dataset
* The dataset used in this project contains customer shopping behavior data.
* It includes information such as customer details, product categories, purchase amounts, and transaction patterns.
* The dataset was loaded and processed using Python.

🛠️ Tools & Technologies
* Python (Pandas) – Data cleaning
* MySQL Server – Data storage and SQL analysis
* Power BI – Dashboard creation and data visualization
* Jupyter Notebook – Development environment

⚙️ Project Steps
1. Data Loading
* Imported dataset into Jupyter Notebook using Pandas
* Checked data structure, columns, and data types

2. Data Cleaning
* Handled missing values
* Removed duplicates
* Standardized column formats
* Corrected inconsistent data

3. SQL Analysis (MySQL)
* Loaded cleaned data into MySQL
* Performed queries such as:
  * Aggregations (SUM, AVG, COUNT)
  * Filtering using WHERE and HAVING
  * Joins and grouping
  * Finding top customers/products

4. Dashboard (Power BI)
* Connected Power BI to MySQL dataset
* Built interactive dashboard including:
  * Sales trends
  * Customer insights
  * Product performance
* Used filters and slicers for better interactivity

📊 Dashboard
* Interactive Power BI dashboard provides a visual summary of key insights
* Helps in quick decision-making
* Includes charts like:
  * Sales by category
  * Monthly trends
  * Top-performing products

📈 Results & Insights
* Identified top-selling products and categories
* Discovered customer purchasing patterns
* Highlighted seasonal sales trends
* Improved data quality through cleaning process
